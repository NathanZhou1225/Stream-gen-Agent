## 一、大盘观点类导流视频（market_view）

### 结构模板（6段，约 60-75 秒）

| 序号 | 段名 | 职责 | 时长 | 必填字段 |
|------|------|------|------|----------|
| 1 | **开篇锁客** | 情绪炸弹+悬念钩子，用冲击性数字或反直觉开场 | 8-12s | `role: hook` / `visual: 情绪词高亮` |
| 2 | **导流/关注** | 展示圈子价值+人设+服务态度，引导加微/关注 | 15-20s | `role: argument_1` / `claim_kind: mixed` / `evidence: 圈子战绩案例` |
| 3 | **亮剑观点** | 一句话结论，让观众记住核心判断 | 8-12s | `role: argument_2` / `claim_kind: opinion` / `visual: 结论字幕` |
| 4 | **论证论点** | 123结构化说服，逻辑+案例让观点站得住脚 | 20-25s | `role: argument_3` / `claim_kind: fact` / `evidence: 数据/板块对比` |
| 5 | **总结强化** | 金句收尾+情绪升华，强化核心价值 | 8-10s | `role: turn` / `visual: 金句字幕` |
| 6 | **行动转化** | 明确指令+利益诱惑，引导关注/加微 | 5-8s | `role: cta` / `cta.type: add_wechat/comment_reply` |

### 段职责详解

**① 开篇锁客（情绪炸弹）**
- 必须用冲击性数字或反直觉开场，例如「2.6万亿！创业板翻绿，行情结束了？」
- 一句话钩住情绪，让观众产生悬念
- 反例：平淡开场「今天聊聊中美会谈」

**② 导流/关注（圈子价值）**
- 必须展示具体战绩案例，例如「4月7号提醒电池、通信，涨幅超10%」
- 强调圈子的价值：实时信号、关键节点提醒
- 给出明确转化入口：评论区蓝链、小助理添加

**③ 亮剑观点（一句话结论）**
- 必须给出明确判断，例如「短线来看，指数回调风险加剧」
- 不模糊、不模棱两可
- 用字幕强化结论词

**④ 论证论点（123结构）**
- 必须用「首先/其次/最后」或「第一/第二/第三」结构
- 每条论据配数据或板块案例
- 例如：「首先，成交放量分歧；其次，高低切换特征；最后，获利盘兑现」

**⑤ 总结强化（金句收尾）**
- 用情绪化金句收尾，例如「轮动是良性调整，后续机会第一时间提醒」
- 强化人设：及时、靠谱、实战派

**⑥ 行动转化（明确指令）**
- 必须给出明确指令：点击评论区链接、添加小助理、进圈子
- 配利益诱惑：免费体验、独家战法、实时信号

## 二、投教知识类导流视频（investor_edu）

### 结构模板（5段，约 60-75 秒）

| 序号 | 段名 | 职责 | 时长 | 必填字段 |
|------|------|------|------|----------|
| 1 | **痛点刺激** | 焦虑唤醒，用痛点话术激活观众需求 | 8-12s | `role: hook` / `visual: 痛点词高亮` |
| 2 | **引导关注** | 展示解决方案价值，引导点关注 | 8-12s | `role: argument_1` / `claim_kind: opinion` / `visual: 关注按钮引导` |
| 3 | **给出方案/干货** | 核心方法论，结构化输出干货内容 | 25-35s | `role: argument_2` / `claim_kind: fact` / `evidence: 方法论步骤` |
| 4 | **深化痛点** | 强化痛点，让观众意识到问题严重性 | 8-12s | `role: turn` / `claim_kind: opinion` / `visual: 痛点对比动画` |
| 5 | **强力钩子** | 回复关键词领资料，转化导流 | 5-8s | `role: cta` / `cta.type: comment_reply` / `cta.phrasing: 回复关键词领取` |

### 段职责详解

**① 痛点刺激（焦虑唤醒）**
- 必须用痛点话术激活需求，例如「还在死扛？还在凭感觉抄底？」
- 用「你亏钱不是运气差，是因为你看不懂」强化痛点
- 反例：平淡开场「今天分享布林带」

**② 引导关注（展示价值）**
- 必须展示方案价值，例如「布林带六字真言免费公开，帮你规避90%坑」
- 引导点关注：干货持续更新

**③ 给出方案/干货（结构化输出）**
- 必须用123结构输出方法论，例如布林带六条规则逐条讲解
- 每条配具体场景判断：上轨之上不言顶、中轨向上强中强等
- 用记忆点：六字真言、简单高效

**④ 深化痛点（强化问题）**
- 必须用对比强化痛点，例如「一买就跌，一卖就涨」
- 用比喻深化：空手打老虎、没有武器

**⑤ 强力钩子（转化导流）**
- 必须给出关键词回复机制，例如「回复「布林」领取实战操盘策略文档」
- 明确利益：精选热门板块资料、实战案例

| 3 | **人设介绍** | 专业背景+独特定位+战法名称 | 12-18s | `role: argument_2` / `claim_kind: fact` / `evidence: 从业年限/战法名称` |
| 4 | **价值展示（续）** | 成功参与板块案例，强化主线能力 | 10-15s | `role: argument_3` / `claim_kind: fact` / `evidence: 板块参与战绩` |

### 段职责详解

**① 开头钩子（战绩钩子）**
- 必须用战绩数据开场，例如「2024年带粉丝躲过5次暴跌抄底3次」
- 用「靠的不是玄学，靠的是独家逃顶技巧」强化专业感

**② 价值展示（板块能力）**
- 必须闪现板块涨幅数据：黄金、纳指、机器人、半导体、银行
- 用「板块投资游刃有余」强化主线能力

**③ 人设介绍（专业背景）**
- 必须给出：从业年限（如18年）、专业定位（逃顶和板块轮动）
- 战法名称：《八度空间战法》
- 江湖人设：「逃顶小王子」

**④ 价值展示（成功案例）**
- 必须列举成功参与的板块：黄金、银行、半导体、机器人
- 强化：提前发现主线、不玩杂毛

**⑤ 社群权益（四大服务）**
- 必须列出四大会员专享服务：
  1. 专业机构消息
  2. 大盘顶底提醒（去年五次逃顶、三次抄底）
  3. 严选板块机会
  4. 独家逃顶战法（开户入金免费获得）
- 差异化权益：一对一陪跑、话聊服务

**⑥ 行动号召（明确入口）**
- 必须给出明确入口：点击左下角「立即咨询」/ 来直播间
- 强化：免费体验、无费用

## 四、通用规则（三种类型共享）

### 制作附录中文段名规则（2026-05-14 新增）

**硬规则**：`production_appendix` 四块（camera_shots / stickers_effects / visual_assets / host_actions）的每条描述必须：
1. **用中文段名开头**：如「开篇锁客」「亮剑结论」「论证三大信号」「关键提醒」「行动转化」，不用英文 role 名称（hook / argument / turn）
2. **贴合大纲段名**：描述内容需对应大纲中实际段名，便于剪辑师快速理解
3. **格式示例**：`开篇锁客：近景推入，数字「2小时」字幕同步出现`（段名 + 具体动作）

**反例**（禁止）：
- `Hook近景推入，数字字幕同步`（英文 role 名称）
- `argument_2近景定格`（技术字段名）

**正例**（必须）：
- `开篇锁客：近景推入，数字「2小时」字幕同步出现`
- `亮剑结论：近景定格，结论字幕「经贸破冰已确认」放大高亮`
- `论证三大信号：中景拍摄，左侧分屏展示三大信号卡片`

此规则适用于所有 content_type（market_view / investor_edu / persona_intro）。

### Hook 必须做到
- 情绪炸弹或痛点刺激，一句完整话（15-25字）
- 不用平淡开场、不用「大家好今天聊聊」

### 论据必须做到
- 每段只讲一个点，配数据/案例/对比
- 用123结构，不堆多个观点

### CTA 必须做到
- 明确指令：点击链接、添加小助理、回复关键词、直播间
- 利益诱惑：免费体验、独家战法、实时信号、精选资料

### 合规红线
- 禁收益承诺：必涨、稳赚、保证收益
- 禁绝对化：必然、绝对、唯一
- 禁荐股暗示：懂的都懂、自己搜、心里有数加代码
- 禁操作指令：今天买、现在卖、抄底
