# CTA 话术库（v1 占位版）

> 🚧 v1 占位版。业务方定稿标准话术后覆盖本文件。

## CTA 类型枚举

v1 支持 3 种 CTA 类型，对应 `script.schema.json` 中 `cta.type` 字段。

### 1. `add_wechat` — 资料钩子（引导加微信）

**场景**：用户对内容中提到的"资料 / 表格 / 清单"有兴趣，引导私信获取。

**默认话术**：

```
想拿今天这张 {topic} 表，评论区扣 "1" 我私发你。
```

**变体**：

```
对 {topic} 有兴趣的朋友，评论区扣 "想要"，我整理好发你。
关注我，私信回复 "{keyword}"，免费领今天说的这份资料。
```

---

### 2. `join_group` — 社群引导

**场景**：引导关注 / 加入财经交流群。

**默认话术**：

```
想看更多这类分析，关注我后进我的财经交流群，每天分享盘面复盘。
```

**变体**：

```
喜欢这种角度的分析，点关注，咱们交流群里见。
关注我，进群一起复盘每天的市场情绪。
```

---

### 3. `dm_keyword` — 私信指令

**场景**：用关键词作为私信暗号，便于批量话术管理。
**默认话术**：

```
对 {topic} 感兴趣的朋友，私信我回复 "{keyword}" 即可领取。
```

**变体**：

```
想要今天这份 {topic_short}，私信我发 "{keyword}"，我秒回。
```

---

## 占位符说明

| 占位符 | 含义 | 示例 |
|---|---|---|
| `{topic}` | 当前稿件核心主题 | "降准复盘" / "北向资金动向" |
| `{topic_short}` | 主题简写 | "降准表" |
| `{keyword}` | 指令暗号 | "降准" / "北向" |

## 位置策略

| `cta.position` | 适用场景 |
|---|---|
| `opening` | 开头就留钩子（较激进，适合信息流） |
| `middle` | 讲完 1 个论据后插入（适合长视频） |
| `ending` | 结尾收单（**默认**，信息密度较集中） |

## 业务方替换指南

部署到业务环境时：

1. 直接覆盖本文件内容，保持章节结构（3 种 CTA 类型、占位符、位置策略）
2. 每种 CTA 至少提供 1 条默认话术、0-N 条变体
3. 不新增 CTA 类型，否则需同步更新 `script.schema.json` 中 `cta.type` 枚举
