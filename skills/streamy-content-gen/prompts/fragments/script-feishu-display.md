# Script Feishu Display（按需读取）

只有在需要把 `script.md` 贴回飞书/聊天给用户看时读取。

## 展示边界

- 读取工具生成的 `script.md`，不要手写 `script.md`。
- 不要用三反引号包整段逐字稿，飞书会显示成代码块。
- 默认只展示逐字稿主体、制作附录、合规状态和下一步动作。
- 不要重复贴 `topic_picking` 的信源状态、大盘行情、快讯、候选依据，除非用户明确要求回看来源。

## 推荐格式

```markdown
#### 逐字稿 #<DID>（约 60 秒）

**[0:00-0:05 · Hook · 贴纸:重点]**
口播正文……

**[0:05-0:22 · 论据 1 · 配图:关键数据卡]**
口播正文……

#### 附录｜详细制作指导

**镜头建议**
- ...

**贴纸/特效**
- ...

**配图建议**
- ...

**人物行为**
- ...

#### 合规状态
扫描通过 / 命中 N 条告警，见工具返回。

修改还是定稿？
```

## 禁止

- 禁止把全文包进 ``` 代码块。
- 禁止展示工具内部 JSON 代替人可读稿。
- 禁止在逐字稿阶段重新展开选题轮的行情与信源全文。
