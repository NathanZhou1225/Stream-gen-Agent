# 财联社信源分项稳定性详解（v0.1）

## 信源范围

- 通过 AkShare 拉取：`stock_info_global_cls(symbol=重点/全部)`
- 在 `news_rss` 中作为快讯补源，与 RSSHub 快讯融合

## 当前角色

- **六大板块快讯增强源**
- 与 RSSHub 形成双源互证（`cross_source_hit`）

## 稳定性结论

- 整体：**中上**
- 依赖 AkShare 环境与字段结构

## 成功点

- 覆盖高时效财经快讯，能显著提升板块密度。
- 与 RSSHub 合并后可提升“同事件多源确认”能力。
- 支持“重点/全部”双池拼接，信息面更广。

## 常见失败

- `CLS_AKSHARE_IMPORT_FAILED`
- AkShare 返回空表（不一定报错）
- 字段名变化导致标题/正文提取失败

## 当前兜底

- 财联社不可用时，快讯主链仍由 RSSHub 提供。
- 财联社失败不影响行情、政策、社媒块。
- 结果层通过去重与排序保障输出一致性。

## 关键建议

1. 固化字段映射兼容（标题/内容/发布时间/发布日期）。
2. 增加“财联社命中率”指标，量化其真实贡献。
3. 将 `cross_source_hit` 比例纳入信源健康看板。
