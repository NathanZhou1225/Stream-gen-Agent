# Tushare 信源分项稳定性详解（v0.1）

## 信源范围

- 北向资金降级：`moneyflow_hsgt`
- 政策增强（可选）：`cctv_news`

## 当前角色

- **北向资金关键降级层**
- AkShare 北向失败时提供数值保底

## 稳定性结论

- 有 token 且接口正常时：**中上**
- 无 token 或配额/网络受限时：**低**

## 成功点

- 北向字段结构明确，能提供可量化净流入值。
- 与主链路解耦：失败只影响北向增强，不影响快照主成功。

## 常见失败

- `NORTHBOUND_TUSHARE_FAILED`
- token 缺失导致不启用（非错误，属于配置未开启）
- 字段单位差异（已做异常大值缩放守护）

## 当前兜底

- Tushare 失败后继续走新浪 7x24 文本抽取和 RSSHub 文本抽取。
- `cctv_news` 是可选补充，不影响政策主链路（证监会/央行官网）。

## 关键建议

1. 生产环境配置并轮换 `TUSHARE_TOKEN`。
2. 对北向单位口径做二次校验（日报对拍）。
3. 将 token 失效与配额不足单独标识，便于运维排障。
